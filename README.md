# Auto Market Price Parser (Sauto.cz to Auto.RIA)

## Description
A Python-based data parsing tool designed to evaluate vehicle import profitability. The script interacts with the Sauto.cz API, retrieves vehicle listings, and maps specific marks and models to compare them with current listings on the Ukrainian platform, Auto.RIA. It automates the calculation of average market prices to identify profitable import opportunities.

## Features
- **API Integration:** Scrapes vehicle data efficiently using the Sauto.cz API.
- **Data Mapping:** Implements a custom mapping logic to match car makes and models between Czech and Ukrainian marketplaces accurately.
- **Price Analysis:** Automatically calculates average market prices on Auto.RIA for the exact mapped models.
- **Profitability Calculation:** Provides data-driven insights into potential margins for importing vehicles.
